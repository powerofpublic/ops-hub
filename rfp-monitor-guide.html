User Guide

Complete guide to searching for opportunities, managing your pipeline, and using every feature.

Power of Public, LLCOperations HubApril 2026

★ Top 5 Things to Know

1
Searching is completely free. The RFP Monitor uses your existing Claude Pro subscription at claude.ai. There is no API key, no per-search cost, and no rate limits. Copy a prompt from the Search tab, paste it into Claude, and paste the results back into the pipeline.
2
Use a different prompt each session. The Search tab has five prompts — each targets a different set of sources. Run 2–3 per session to get the broadest coverage. Never use the same prompt two weeks in a row without rotating.
3
Search on Tuesday or Wednesday morning. Government agencies post solicitations Monday–Tuesday. Mid-week searches catch everything from the previous week with time left to respond.
4
Check the Manual Portals tab every Thursday. Many of the best sources require a login and cannot be searched automatically. The portal checklist tracks which sites you've reviewed each week and resets every Monday.
5
Mark opportunities "Considering / Go No-Go" early. This status adds a direct link to your Go/No-Go evaluation form on the pipeline card, keeping your decision process connected to the opportunity.

★ Best Day & Weekly Rhythm

Tuesday or Wednesday morning is the ideal time to run your weekly searches.

Government procurement offices post solicitations Monday–Tuesday. Foundations and nonprofits tend to post mid-week. Searching Tuesday or Wednesday catches everything posted since the previous week — with enough time left in the week to respond to anything promising.

DayAction
Tue / Wed morningOpen the Search tab → copy 2–3 prompts → paste each into Claude Pro → paste results back into Import from Claude
ThursdayOpen the Manual Portals tab → work through the checklist, reviewing login-required sites → update pipeline opportunity statuses
FridayGo to the Email Digest tab → click Generate & Copy Digest → paste into email or save as a weekly record

★ Prompt Rotation Strategy

The Search tab has five prompts, each targeting a different set of sources. Rotating through them across sessions gives you the broadest possible coverage without repeating the same sources week after week.

PromptBest used when

Communications-SpecificAlternate weeks — O'Dwyer's and rfpdb.com are good for niche comms RFPs
Regional — Maryland & DCAlternate weeks — focus on this when pursuing state and local work

Save your most-used prompts to the Saved Searches feature in the Email Digest tab so you can quickly reapply them to filter your pipeline later.

Import from Claude

The Import from Claude box sits at the top of the Pipeline tab. After running a search in Claude Pro, paste the full response and click + Add all to pipeline. The importer handles both JSON and plain formatted text.

  1. Go to the Search tab → click Copy prompt
  2. Open claude.ai → paste the prompt → run the search
  3. Copy Claude's full response
  4. Return to Pipeline tab → click Show / Hide on the Import box
  5. Paste → click + Add all to pipeline
Duplicate detection is built in — if an opportunity with the same title already exists in your pipeline, it is silently skipped on import.

Notes

Click 📝 Add notes on any card to record strategy thinking, contacts, or next steps. Notes are saved automatically, display on the card, and are included in the weekly digest export.

Value Tracking

Enter an estimated contract value when adding or editing an opportunity. The Pipeline $ stat in the header bar sums all values across your active pipeline automatically.

Duplicate Detection

  • On import — opportunities matching an existing title are silently skipped
  • On manual add — an alert appears before adding a duplicate title

Weekly Checklist

Every portal card has a checkbox. Check it off after reviewing that site. The progress counter at the top shows "12 / 49 portals reviewed this week." The checklist resets automatically each Monday. Click Reset week to clear it manually at any time.

Use Thursday as your manual portal review day — work through the login-required sections that Claude's web search cannot access automatically.

Adding or Removing Portals

Portals are permanently stored in the HTML file — not your browser — so they appear consistently on all devices and are never lost when the file is updated. To add or remove a portal:

  1. Ask Claude: the portal name, URL, and which section it belongs to
  2. Download the updated rfp-monitor.html
  3. Upload to your ops-hub GitHub repository

4 Win / Loss Tab

Your business development scorecard. Automatically tracks the outcomes of opportunities moved to Won, Lost, or No-Go status in the Pipeline tab. Shows win rate percentage, total value won, and a full outcome log.

Win/Loss vs. Go/No-Go form: The Go/No-Go form helps you decide whether to pursue an opportunity (before bidding). The Win/Loss tab records what happened after you submitted (after the client decides). They serve different stages of the same process.

The tab updates automatically as you change statuses in the Pipeline — no manual data entry needed.

5 Email Digest Tab

Two tools in one tab: a weekly pipeline export and a saved searches manager.

Weekly Digest Export

Click Generate & Copy Digest to build a formatted plain-text summary of your full pipeline. The digest includes:

  • Pipeline summary — active opportunity count and total estimated value
  • Urgent deadlines — anything due in the next 14 days, sorted by date
  • Opportunities grouped by status (New, Considering, Pursuing, Submitted)
  • Notes displayed under each opportunity
  • Won opportunities this cycle

The digest copies to your clipboard automatically and also appears in a text box for manual copy. Paste into an email, a Word document, or anywhere you track business development activity.

Saved Searches

Type a keyword combination and click Save to store it. Up to 5 searches can be saved. Click any saved search to instantly apply it as a filter in the Pipeline tab. Click ✕ to delete a saved search.

Save your 5 most productive keyword combinations here so you can rotate through them each week without retyping.

★ Adding or Removing Sources

All sources and portals are permanently stored in the HTML file — not your browser. To add or remove any source or portal, ask Claude: share the name, URL, and type (Government / Foundation / Nonprofit / Academic). Download the updated file and upload it to your ops-hub GitHub repository.

★ Troubleshooting

IssueFix
Page is blank or won't loadHard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
Tabs don't switchHard refresh the page. If it persists, re-upload the latest rfp-monitor.html to GitHub.
Claude returns no resultsTry a different prompt or broader keywords. Run 2–3 different prompts per session.
Import from Claude adds nothingPaste the full Claude response — all text, not just part of it. Both JSON and plain text are supported.
Portal links don't openCheck your browser isn't blocking popups. All links open in a new tab.
Portal checklist resets unexpectedlyThe checklist resets automatically each Monday. Use Reset week to clear manually.
Digest not copying to clipboardThe text also appears in the box below the button — copy it from there.
Pipeline data disappearedPipeline data is stored in browser localStorage — it is device-specific. If you clear browser data it resets. Export the digest weekly as a backup record.
Site not updating after GitHub uploadWait 60 seconds for GitHub Pages to rebuild, then hard refresh.
