About Manual Portals
What these are: Sites that require a login or pass a reCAPTCHA check and cannot be searched automatically. Bookmark this tab for your weekly manual check — all links open in a new tab. Most have a public browsing mode; you only need to log in to submit a response.

How to add or remove a portal: Portals are permanently stored in the HTML file — not your browser. This keeps them consistent across all devices and safe from browser data clears. To add or remove one, ask Claude: share the portal name, URL, and which section it belongs to (State procurement, DC-adjacent, PR lists, Maryland Board of Public Works, or My portals). Upload the updated file to GitHub and it's live everywhere instantly.
